# Решение проблемы с 3D моделями

## Важно!
Для работы сайта **НЕОБХОДИМ** локальный сервер. Браузеры блокируют загрузку файлов через `file://` протокол из-за безопасности (CORS).

## Быстрый запуск сервера:

### Windows (через Python):
1. Откройте командную строку (cmd) или PowerShell
2. Перейдите в папку проекта:
   ```
   cd C:\Users\User\Downloads\site\uchebnik.kz
   ```
3. Запустите сервер:
   ```
   python -m http.server 8000
   ```
4. Откройте браузер: http://localhost:8000

### Или используйте start-server.bat:
Просто запустите файл `start-server.bat` двойным кликом.

## Если модели все еще не загружаются:

1. **Откройте консоль браузера (F12)** и проверьте ошибки
2. **Убедитесь, что используете локальный сервер** (не открывайте файл напрямую!)
3. **Проверьте, что файлы .glb находятся в папке `models/`**
4. **Проверьте имена файлов** - они должны точно совпадать с именами в `js/main.js`
