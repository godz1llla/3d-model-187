# Образовательный сайт "Задачи по достопримечательностям Кызылорды"

Статический образовательный веб-сайт с интерактивными математическими задачами и 3D моделями.

## Структура проекта

```
/project-root
├── index.html                # Главная страница
├── object.html               # Шаблон страницы объекта
├── css/
│   └── style.css            # Все стили и анимации
├── js/
│   ├── main.js              # Навигация и анимации
│   ├── lang.js              # Система управления языками
│   ├── tasks.js             # Логика задач и проверка ответов
│   └── viewer3d.js          # Three.js просмотрщик GLB моделей
├── data/
│   ├── common-kz.txt        # Общие переводы (казахский)
│   ├── common-ru.txt        # Общие переводы (русский)
│   ├── common-en.txt        # Общие переводы (английский)
│   ├── kz/
│   │   ├── hero.txt         # Текст главной страницы
│   │   ├── objects-list.txt # Список названий объектов
│   │   ├── object1.txt      # Контент объекта 1
│   │   ├── object2.txt      # Контент объекта 2
│   │   └── ... (object3-15.txt)
│   ├── ru/
│   │   └── ... (та же структура)
│   └── en/
│       └── ... (та же структура)
├── models/                   # 3D модели (.glb файлы)
│   ├── Центр искусств.glb
│   └── ... (остальные .glb файлы)
└── images/                   # Изображения объектов
    ├── object1/
    │   └── list.txt         # Список изображений
    └── ... (папки для других объектов)
```

## Формат файлов данных

### Формат object*.txt:

```
[TITLE]
Название объекта

[DESCRIPTION]
Описание объекта. Может быть несколько абзацев.
Каждый абзац на новой строке.

[TASK]
Текст задачи. Здесь описание математической задачи.

[ANSWER]
Правильный ответ (например: 28° и 28°)
```

### Формат common-*.txt:

```
[back]
← Назад
[model-title]
3D Модель
[fullscreen]
Полноэкранный режим
[task-title]
Задача
[answer-placeholder]
Введите ответ...
[check-btn]
Проверить ответ
[footer-text]
© 2026 Образовательный сайт
```

### Формат hero.txt:

```
Первая строка - заголовок главной страницы
Вторая строка - вводный текст
Третья строка - заголовок раздела объектов
```

### Формат objects-list.txt:

Один объект на строку, в том же порядке, что и в OBJECTS массиве в main.js

## Как добавить контент

### 1. Добавить новый объект:

1. Добавьте запись в массив `OBJECTS` в `js/main.js`
2. Создайте файлы `data/kz/objectN.txt`, `data/ru/objectN.txt`, `data/en/objectN.txt`
3. Добавьте название объекта в `data/*/objects-list.txt`
4. Добавьте 3D модель в папку `models/`
5. (Опционально) Добавьте изображения в `images/objectN/`

### 2. Добавить изображения для объекта:

1. Создайте папку `images/objectN/`
2. Поместите изображения в эту папку
3. Создайте файл `images/objectN/list.txt` со списком имен файлов (по одному на строку)

### 3. Добавить 3D модель:

1. Поместите .glb файл в папку `models/`
2. Обновите поле `modelFile` в массиве `OBJECTS` в `js/main.js`

## Запуск сайта

Это статический сайт, но он требует локальный сервер из-за CORS политики браузера.

### Вариант 1: Python

```bash
python -m http.server 8000
```

Затем откройте в браузере: `http://localhost:8000`

### Вариант 2: Node.js (если установлен)

```bash
npx http-server
```

### Вариант 3: VS Code Live Server

Установите расширение "Live Server" в VS Code и нажмите "Go Live"

## Технологии

- **HTML5** - структура
