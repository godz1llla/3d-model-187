# Инструкция по запуску сайта

## Важно! Сайт должен запускаться через локальный сервер

Сайт **НЕ будет работать**, если открывать HTML файлы напрямую (двойной клик по файлу), потому что браузер блокирует загрузку файлов через `fetch()` из-за политики CORS.

## Как запустить сайт:

### Вариант 1: Python (рекомендуется)

1. Откройте командную строку (PowerShell или CMD) в папке проекта
2. Запустите команду:

```bash
python -m http.server 8000
```

3. Откройте в браузере: `http://localhost:8000`

### Вариант 2: Node.js (если установлен)

```bash
npx http-server
```

Затем откройте адрес, который покажет команда (обычно `http://localhost:8080`)

### Вариант 3: VS Code Live Server

1. Установите расширение "Live Server" в VS Code
2. Откройте папку проекта в VS Code
3. Правой кнопкой на `index.html` → "Open with Live Server"

## Проверка

После запуска сервера:
- Главная страница: `http://localhost:8000/index.html`
- Страница объекта: `http://localhost:8000/object.html?id=1`

## Устранение проблем

Если данные не загружаются:
1. Убедитесь, что используете локальный сервер (не file://)
2. Проверьте консоль браузера (F12) на наличие ошибок
3. Убедитесь, что все файлы данных созданы в папке `data/`
